# Quest Manager - Геймифікований Менеджер Завдань 🎮

RPG-стиль система управління завданнями з повною українською локалізацією.

## Особливості MVP

### Базові функції (доступні з 1-го рівня):
- ✅ **Система персонажа**: Створення персонажа з ім'ям, рівнем, XP, монетами
- ✅ **Управління завданнями**: Створення щоденних завдань з різними рівнями складності
- ✅ **Система прогресу**: Виконання завдань, отримання XP та монет
- ✅ **Статистика**: Відображення виконаних завдань та відсотку виконання
- ✅ **Теми**: Перемикання між темною та світлою темами
- ✅ **Повна українізація**: Весь інтерфейс українською мовою

### Розширені функції (розблокуються на 3-му рівні):
- 🔒 Система звичок з трекером
- 🔒 Багатокрокові квести
- 🔒 8 категорій навичок (Фітнес, Навчання, Робота, Побут, Творчість, Здоров'я, Фінанси, Соціальне)
- 🔒 Система досягнень
- 🔒 Магазин (аватари, титули, підсилення)
- 🔒 Цілі та віхи
- 🔒 Конструктор квестів
- 🔒 Детальна аналітика
- 🔒 Серії (streaks) та щоденні виклики
- 🔒 Лідерборди

## API Endpoints

### Авторизація
- `POST /api/auth/register` - Реєстрація з ім'ям персонажа
- `POST /api/auth/login` - Вхід

### Користувач
- `GET /api/user/me` - Інформація про поточного користувача
- `POST /api/user/allocate-stats` - Розподіл очок характеристик
- `POST /api/user/theme` - Зміна теми

### Завдання
- `GET /api/tasks` - Список завдань
- `POST /api/tasks` - Створення завдання
- `POST /api/tasks/{id}/complete` - Виконання завдання
- `DELETE /api/tasks/{id}` - Видалення завдання

### Статистика
- `GET /api/stats/overview` - Огляд статистики

## Система XP та Рівнів

### Нарахування XP:
- **Легко**: 15 XP, 10 монет
- **Середньо**: 35 XP, 25 монет
- **Важко**: 75 XP, 50 монет

### Прогрес рівнів:
- Формула: XP_needed = 100 × рівень × 1.5
- При підвищенні рівня: +3 очки характеристик
- Максимальний рівень: 100

### Характеристики:
- 💪 Сила (Strength)
- 🧠 Інтелект (Intelligence)
- ❤️ Витривалість (Stamina)
- ⚡ Спритність (Agility)
- 🎨 Креативність (Creativity)
- 👥 Харизма (Charisma)
